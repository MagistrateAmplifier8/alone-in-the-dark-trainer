# Alone in the Dark Trainer

A trainer for the classic game *Alone in the Dark* (1992). Provides memory-based cheats for health, sanity, inventory, and more.

## Features
- Unlimited Health
- Unlimited Sanity
- Infinite Ammo
- Set Inventory Items (Key, Gun, etc.)
- Game Speed Control
- Teleport to Checkpoint

## Requirements
- .NET 6.0 or later
- Windows (uses WinAPI for memory manipulation)

## Build
```
dotnet build
```

## Test
```
dotnet test
```

## Usage
Run the trainer alongside the game. Use hotkeys to toggle cheats.
